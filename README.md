# 3x3-lifegame
### lifegame.c 3x3 ライフゲーム  
   Linux環境では
   ```
   cc lifegeme.c
   ```
   でコンパイル後
   ```
   ./a.out
   ```
   で起動。例えば
   ```
   input a numbers of 1 row
   1 1 1
   input a numbers of 2 row
   0 1 0
   input a numbers of 3 row
   1 1 1
   ```
   と入力してみると?  
   しかし
   ```
   input a numbers of 1 row
   0 0 0 
   input a numbers of 2 row
   0 1 1
   input a numbers of 3 row
   1 0 1
   ```
   では?(グライダー)
